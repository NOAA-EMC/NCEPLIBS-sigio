# SIGIO

API to perform I/O from the spectral GFS model

Code Manager: George Vandenberghe


### Prerequisites

Compilers: GNU | Intel | Clang | AppleClang 


### Installing

```
mkdir build
cd build
cmake -DCMAKE_INSTALL_PREFIX=/path/to/install /path/to/NCEPLIBS-sigio
make -j2
make install
```
